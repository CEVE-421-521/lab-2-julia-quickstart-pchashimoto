Defining a function

In index.qmd, we read in a CSV file from scratch. However, weâ€™d like to repeat this process for each year of data, and to do it in a consistent way so that we can read in the data for all available years into a single file. To do this, weâ€™ll write a function that we can use to read in the data for any year. Specifically, our function will take in the year as an argument, and return a DataFrame with the data for that year.

Before we do that, letâ€™s define a function that will return the filename for a given year. Itâ€™s often valuable to stack several functions together.

get_fname(year::Int) = "data/tidesandcurrents-8638610-$(year)-NAVD-GMT-metric.csv"

Now weâ€™re ready to define our function:

function read_tides(year::Int)
    
    # define the CSV file corresponding to our year of choice
    fname = get_fname(year)

    # a constant, don't change this
    date_format = "yyyy-mm-dd HH:MM"
    
    # 1. read in the CSV file and save as a dataframe
    # 2. convert the "Date Time" column to a DateTime object
    df = CSV.read(fname, DataFrame; dateformat = date_format)
    
    # 3. convert the " Water Level" column to meters
    df[!, " Water Level"] .*= 1u"m" # uses Unitful package

    # 4. rename the columns to "datetime" and "lsl"
    df = @rename(df, :datetime = $"Date Time", :lsl = $" Water Level")

    # 5. select the "datetime" and "lsl" columns
    df = @select(df, :datetime, :lsl)

    # 6. return the dataframe
    return df 
end
